# 🎓 EducaIA

Plataforma inteligente de análise acadêmica desenvolvida com Streamlit.

## Funcionalidades

- Dashboard analítico
- KPIs educacionais
- Georreferenciamento
- Inteligência analítica
- Relatórios dinâmicos
- Simulação de IA
- Visualizações avançadas

## Tecnologias

- Python
- Streamlit
- Plotly
- Pandas
- Folium

## Como executar

```bash
pip install -r requirements.txt
streamlit run app.py
